# 정렬 알고리즘 시간복잡도 비교

- ### 시간 복잡도  

  |                  Sort Algorithm                   |                T(n)                |
  | :-----------------------------------------------: | :--------------------------------: |
  | Bubble sort<br/>Insertion sort<br/>Selection sort |               O(n^2)               |
  |                    Quick sort                     | 최악 : O(n^2)<br/>평균 : O(nlog2n) |
  |                    Merge sort                     |             O(nlog2n)              |
  |                     Heap sort                     |             O(nlog2n)              |
  |           Counting sort<br/>Radix sort            |        O(n+k)<br/>O(d(n+k))        |
